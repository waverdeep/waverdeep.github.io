WaveBYOL: Self-Supervised Learning for Audio Representation From Raw Waveforms

레이블 없는 대규모 오디오 데이터로부터 범용적인 표현(representation)을 학습하기 위해, BYOL 프레임워크를 raw waveform에 직접 적용한 자기지도학습 모델. mel-spectrogram으로의 전처리 단계를 제거해 시간영역 정보 손실 없이 end-to-end로 학습 가능.

02 Motivation

딥러닝 기반 오디오 모델의 성능은 대규모 레이블에 크게 의존해 왔고, 분야별로 레이블을 구축하는 비용이 높다는 한계가 있었습니다. 이를 해소하기 위한 자기지도학습(Self-supervised Learning)은 이미지 도메인에서 SimCLR·MoCo·BYOL 계열로 빠르게 발전했지만, 오디오 쪽은 여전히 mel-spectrogram을 입력으로 사용하는 방식이 주류였습니다.

그러나 mel-spectrogram은 수동적으로 설계된 전처리 파이프라인이며, STFT·mel filter bank를 거치는 과정에서 시간영역 정보 일부가 손실됩니다. 이 연구는 다음과 같은 질문에서 출발했습니다.

"Raw waveform만으로 레이블 없이, 범용적인 오디오 표현을 학습할 수 있을까?"

BYOL은 negative sample 없이도 무너지지 않는(non-collapsing) 표현을 학습할 수 있다는 장점이 있어, raw waveform처럼 배치 구성 비용이 큰 도메인에 특히 적합하다고 판단했습니다.

03 Approach

아키텍처 개요

WaveBYOL은 BYOL의 online–target 비대칭 구조를 그대로 계승하되, 입력을 mel-spectrogram이 아닌 raw waveform으로 두고 전 구간을 end-to-end로 학습합니다. 동일한 waveform에 두 가지 서로 다른 augmentation을 적용해 view 쌍을 만들고, online network의 예측(prediction)이 EMA로 갱신되는 target network의 projection에 가깝도록 학습합니다.

  • Encoder — raw waveform을 직접 받아 고차원 representation으로 매핑하는 1D CNN 기반 encoder
  • Projector / Predictor — BYOL와 동일한 MLP projector 및 online-only predictor
  • EMA target — online network 파라미터의 지수이동평균으로 target network 갱신
  • Loss — 두 view의 projection 간 L2 normalized MSE (contrastive-free)

Raw Waveform 특화 Augmentation

Mel-spectrogram 기반 기법에 쓰이던 SpecAugment 계열은 raw waveform에 그대로 적용할 수 없기 때문에, 시간영역에서 의미를 유지하면서 view 간 다양성을 만들어주는 waveform-level augmentation 조합을 설계했습니다. 대표적으로 다음 요소를 사용했습니다.

  • Random cropping / time shifting
  • Gaussian noise, background noise mixing
  • Gain / polarity inversion
  • Pitch shift · time stretch
  • Low/high-pass filtering

각 augmentation의 조합과 적용 확률은 downstream 평가 성능을 기준으로 ablation을 통해 결정했습니다.

04 Key Contributions

  • Raw waveform 기반 BYOL 프레임워크 제안 — mel-spectrogram 전처리를 제거하고 시간영역 정보 손실 없이 end-to-end로 표현을 학습하는 자기지도학습 모델을 설계
  • Waveform 전용 Augmentation 조합 설계 — raw waveform에 적합한 다양성 확보 전략을 실험적으로 정립 (ablation 제공)
  • Negative-free 학습의 오디오 적용성 검증 — 대규모 배치를 요구하는 contrastive 계열의 한계를 우회하며 경쟁력 있는 표현을 획득
  • 다양한 Downstream 태스크 일반화 — audio event classification, speech command, music classification 등 여러 태스크에서 linear-probe 성능으로 범용성 평가

05 Experiments

Pre-training

  • 대규모 비레이블 오디오 코퍼스에서 WaveBYOL encoder를 사전학습
  • Optimizer·lr schedule·EMA decay 등 주요 하이퍼파라미터는 원 BYOL 세팅을 기반으로 오디오 도메인에 맞춰 조정

Downstream 평가

Encoder를 freeze한 상태에서 downstream 태스크별로 linear classifier만 학습하는 linear-probe 프로토콜로 범용성을 측정했습니다. 비교군으로는 동일 조건에서의 mel-spectrogram 기반 SSL 베이스라인을 설정했습니다.

  • Audio event classification (예: ESC / UrbanSound 계열)
  • Speech command classification (예: Speech Commands)
  • Music / instrument classification

세부 수치와 ablation 결과는 IEEE Access 게재본 Section IV를 참고해 주세요.

06 Takeaways

WaveBYOL 연구를 통해 다음과 같은 경험을 얻었습니다.

  • 이미지 도메인의 SSL 아이디어(BYOL)를 다른 modality(raw audio)로 옮길 때의 설계 고려사항 — 입력 표현, augmentation, encoder 구조의 재설계 과정
  • 대규모 비레이블 데이터를 어떻게 학습 신호로 전환할지에 대한 실전 감각 — 이후 NEOWIZ에서 LLM·Embedding 기반 검색/분석 시스템을 설계할 때의 토대가 됨
  • 연구 주제를 선정·설계하고 SCIE 저널(1저자)로 출판하기까지의 end-to-end 사이클 경험
